Color Memory Site Plan

Kaden Layland

WDD 131-02

Overview

Purpose

To create a fun memory game with colors to show off the principles that I learned in this class.

Audience

My audience is those who like to play games online. This could be children or teenagers who are tired of the monotony of school. This could also be people who enjoy video games

Dynamic elements

Javascript will use float elements to randomly order cards that are flipped over. Each card will have a matching color. The player will have to find its match by remembering where each color is. The player only has a certain ammount of mistakes. Javascript will be used to randomize and impliment these elements onto the html.
